# Diker Straßenbau Website

## Über das Projekt

Professionelle Website für Diker Straßenbau GmbH - Ihr Straßenbauunternehmen in Solingen.

## Technologien

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Lokale Entwicklung

```sh
# Repository klonen
git clone <YOUR_GIT_URL>

# In das Projektverzeichnis wechseln
cd <YOUR_PROJECT_NAME>

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev
```
